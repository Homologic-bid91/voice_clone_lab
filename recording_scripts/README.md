# Recording Scripts

Use `voice_dataset_reading_script.txt` as the source text for clean voice recordings.

| Target | Read |
|--------|------|
| About 5 minutes | BLOCK 01 through BLOCK 05 |

The timing assumes a slow, clear teaching pace with short pauses. If you read quickly, continue into more blocks until the cleaned audio reaches your target duration.
